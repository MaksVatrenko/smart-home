# Auth Page Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Добавить страницу авторизации/регистрации с mock-аутентификацией через Pinia и middleware-защитой всех существующих страниц.

**Architecture:** Pinia store `auth.ts` хранит сессию и персистирует её в `localStorage` вручную. Route middleware `auth.ts` редиректит неавторизованных пользователей на `/login`. Страница `/login` использует отдельный layout `auth.vue` без footer/modal и содержит два таба с компонентами форм.

**Tech Stack:** Nuxt 4, Vue 3, Pinia, Vue i18n, SCSS (с автоимпортом abstracts).

## Global Constraints

- Package manager: **pnpm**
- Путь к исходникам: `app/` (алиас `@`)
- SCSS abstracts (`_variables`, `_mixins`, `functions`) доступны в каждом `<style lang="scss">` без явного импорта
- i18n ключи добавляются в оба файла: `app/locales/ua.json` и `app/locales/en.json`
- Стиль карточки модального окна: `border: 1px solid rgba(90, 51, 255, 0.42)` + box-shadow из `app/components/layout/Modal.vue:82-87`
- Кнопки сабмита: градиент `#5b6bff → #8b5cff`, `border-radius: em(12)`, белый текст
- Цвет ошибок валидации: CSS-переменная `$color-red`
- Mock-правило успеха: email не пустой + `password.length >= 6`
- Нет реального API — только `localStorage`

### Task 1: Pinia auth store

**Files:**
- Create: `app/stores/auth.ts`

**Interfaces:**
- Produces:
  - `useAuthStore()` — composable Pinia store
  - `state.isAuthenticated: boolean`
  - `state.user: { name: string; email: string } | null`
  - `actions.login(email: string, password: string): boolean` — возвращает `true` при успехе
  - `actions.register(name: string, email: string, password: string): boolean` — возвращает `true` при успехе
  - `actions.logout(): void`
  - `actions.restoreSession(): void`

- [ ] **Шаг 1: Создать файл store**

```ts
// app/stores/auth.ts
import { defineStore } from 'pinia'

const STORAGE_KEY = 'smarthome_auth'

export const useAuthStore = defineStore('auth', {
  state: () => ({
    user: null as { name: string; email: string } | null,
    isAuthenticated: false
  }),

  actions: {
    login(email: string, password: string): boolean {
      if (!email || password.length < 6) return false
      this.user = { name: email.split('@')[0], email }
      this.isAuthenticated = true
      localStorage.setItem(STORAGE_KEY, JSON.stringify(this.user))
      return true
    },

    register(name: string, email: string, password: string): boolean {
      if (!name || !email || password.length < 6) return false
      this.user = { name, email }
      this.isAuthenticated = true
      localStorage.setItem(STORAGE_KEY, JSON.stringify(this.user))
      return true
    },

    logout(): void {
      this.user = null
      this.isAuthenticated = false
      localStorage.removeItem(STORAGE_KEY)
      navigateTo('/login')
    },

    restoreSession(): void {
      const raw = localStorage.getItem(STORAGE_KEY)
      if (raw) {
        try {
          this.user = JSON.parse(raw)
          this.isAuthenticated = true
        } catch {
          localStorage.removeItem(STORAGE_KEY)
        }
      }
    }
  }
})
```

```ts
// app/middleware/auth.ts
export default defineNuxtRouteMiddleware(() => {
  const auth = useAuthStore()
  auth.restoreSession()
  if (!auth.isAuthenticated) {
    return navigateTo('/login')
  }
})
```

**Files:**

- [ ] **Шаг 1: Создать компонент**

```vue
<!-- app/components/auth/LoginForm.vue -->
<template>
  <form class="login-form" @submit.prevent="handleSubmit">
    <div class="login-form__field">
      <FormsInput
        id="login-email"
        v-model="email"
        type="email"
        :label="t('auth.email')"
        :placeholder="t('auth.email')"
      />
      <span v-if="errors.email" class="login-form__error">{{ errors.email }}</span>
    </div>

    <div class="login-form__field">
      <FormsInput
        id="login-password"
        v-model="password"
        type="password"
        :label="t('auth.password')"
        :placeholder="t('auth.password')"
      />
      <span v-if="errors.password" class="login-form__error">{{ errors.password }}</span>
    </div>

    <button type="submit" class="login-form__btn">
      {{ t('auth.login') }}
    </button>
  </form>
</template>

<script setup lang="ts">
import { useI18n } from 'vue-i18n'

const { t } = useI18n()
const auth = useAuthStore()
const emit = defineEmits<{ success: [] }>()

const email = ref('')
const password = ref('')
const errors = reactive({ email: '', password: '' })

function validate(): boolean {
  errors.email = ''
  errors.password = ''
  let valid = true
  if (!email.value) { errors.email = t('auth.required'); valid = false }
  if (password.value.length < 6) { errors.password = t('auth.minPassword'); valid = false }
  return valid
}

function handleSubmit() {
  if (!validate()) return
  const ok = auth.login(email.value, password.value)
  if (ok) emit('success')
}
</script>
```
